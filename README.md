# Online-Retail-Sales-Analysis-PowerBI
End-to-end Sales Analysis project using Excel and Power BI. Includes data cleaning, data modeling, DAX measures, KPI analysis, customer insights, product performance, and interactive dashboards built from Online Retail data
The project demonstrates the complete data analytics workflow: Data Cleaning in Excel,Data Modeling in Power BI,DAX MeasureCreation,Dashboard Design,Business Insights Generation
Objectives: Analyze overall sales performance,Identify top-performing products,Understand customer purchasing behavior,Track monthly and yearly sales trends,Compare sales across different countries,Measure customer retention through repeat purchases
Tools Used: Microsoft Excel,Power BI Desktop,DAX,Power Query Editor
Dataset Information : Invoice Number,Product Details,Quantity,Unit Price,Customer ID,Country,Invoice Date
The following cleaning steps were performed for data cleaning: Removed duplicate records,Handled missing values,Removed cancelled orders,Corrected data types,Created date hierarchy,Standardized country names
Data Model Star Schema Design
Dashboard Pages 1. Sales Performance Dashboard 2. Customer Analysis Dashboard
Sales Insights,strong sales growth observed during the final quarter, a few products contribute significantly to overall revenue,monthly trends reveal seasonal fluctuations,over 400 customers made repeat purchases,most customers are concentrated in the United Kingdom,high-value customers contribute a large percentage of revenue,united Kingdom dominates sales volume,Australia and several European countries are strong secondary markets
